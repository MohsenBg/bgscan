# bgscan

**bgscan** is a powerful terminal-based multi-protocol scanner built in Go. It probes IPs and ranges using ICMP, TCP, DNS, HTTP (including HTTP/3), and Xray protocols and all through an intuitive keyboard-driven terminal UI.

Whether you're testing network connectivity, finding optimal endpoints, or scanning for Xray proxies, bgscan provides fast, reliable results with minimal resource usage.
