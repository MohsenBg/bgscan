# شروع کار با bgscan

به **bgscan** خوش آمدید! این راهنما یک دید کلی و سریع از چیزهایی که بعد از نصب با آن‌ها روبه‌رو می‌شوید و نحوه اجرای اولین اسکن را به شما نشان می‌دهد.

## اجرای bgscan

به پوشه محل نصب برنامه بروید و آن را اجرا کنید:

- **ویندوز:** روی فایل `bgscan.exe` دو بار کلیک کنید یا دستور `.\bgscan.exe` را در پاورشل اجرا کنید.
- **لینوکس/مک/ترموکس:** دستور `./bgscan` را در ترمینال خود بزنید.

## آشنایی با منوی اصلی

وقتی bgscan را اجرا می‌کنید، منوی اصلی با گزینه‌های زیر پیش روی شما قرار می‌گیرد:

| گزینه | توضیحات |
|--------|-------------|
| **Run Scan** | شروع یک اسکن شبکه جدید |
| **IP Files** | مدیریت و انتخاب فایل‌های لیست آی‌پی برای اسکن |
| **Result Files** | مشاهده و مدیریت نتایج ذخیره‌شده اسکن‌ها |
| **Settings** | پیکربندی پارامترها و تنظیمات دلخواه اسکن |
| **Xray Outbounds** | مدیریت کانفیگ‌های پروکسی Xray |
| **Logs** | مشاهده لاگ‌های برنامه و اسکن‌ها |

## کلیدهای کنترل و ناوبری

از این کلیدهای میانبر برای جابه‌جایی در برنامه استفاده کنید:

- **کلیدهای جهت‌نما** (↑ ↓) — حرکت به بالا و پایین بین گزینه‌های منو
- **Enter** — انتخاب گزینه هایلایت‌شده
- **b** یا **Esc** — بازگشت به صفحه قبلی
- **q** — خروج از برنامه

> 💡 **نکته:** هر زمان که خواستید، با فشردن کلید `q` می‌توانید با خیال راحت از bgscan خارج شوید.

## اجرای اولین اسکن (شروع سریع)

چطور در کمتر از ۲ دقیقه یک اسکن پایه را اجرا کنیم:

۱. ابزار **bgscan** را از پوشه محل نصب اجرا کنید.

۲. گزینه **Run Scan** را انتخاب کرده و Enter را بزنید.

۳. یک لیست آی‌پی انتخاب کنید (برای شروع می‌توانید از لیست‌های داخلی مثل `cloudflare_IPv4` استفاده کنید).

۴. نوع اسکن را مشخص کنید (پروتکل `tcp` برای یک تست اتصال سریع پیشنهاد می‌شود).

۵. برای شروع اسکن، کلید Enter را بزنید.

۶. منتظر بمانید تا فرآیند اسکن کامل شود.

۷. نتایج را روی صفحه بررسی کنید؛ همچنین بعداً می‌توانید از منوی اصلی وارد بخش Result Files شوید و آن‌ها را مرور کنید.

## گام بعدی چیست؟

حالا که با پایه‌ای‌ترین بخش‌ها آشنا شدید:

- سری به بخش **Settings** بزنید تا رفتار اسکن‌ها را شخصی‌سازی کنید.
- انواع مختلف اسکن را در بخش **Run Scan** امتحان کنید.
- در بخش **IP Files** یاد بگیرید که چطور لیست آی‌پی‌های اختصاصی خودتان را وارد کنید.
- برای آشنایی با ویژگی‌های پیشرفته‌تر، [راهنمای کامل کاربری](../scanner/) را مطالعه کنید.
