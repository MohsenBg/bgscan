# راهنمای توسعه‌دهندگان (Developer Guide)

این بخش برای کسانی است که می‌خواهند با ساختار و معماری داخلی bgscan آشنا شوند، در توسعه پروژه مشارکت کنند یا آن را از روی سورس‌کد کامپایل و بنا کنند.

- [شروع به کار](./getting-started/) — کلون کردن، کامپایل و اجرای محلی پروژه
- [معماری](./architecture/) — ساختار کلی پروژه و نحوه اتصال لایه‌های مختلف به یکدیگر
- [هسته اصلی](./core/) — موتور اسکنر، اینترفیس پروب‌ها، سیستم پیکربندی و خط‌لوله نتایج
- [رابط کاربری](./ui/) — معماری TUI (رابط کاربری متنی)، مدل کامپوننت‌ها، چیدمان و تم‌ها
- [مشارکت در پروژه](./contributing/) — سیستم برانچ‌بندی، استایل کامیت‌ها و گردش‌کار پول‌ریکوئست‌ها
