# تنظیمات ICMP (ICMP Settings)

> 💡 **نکته:** شما می‌توانید این تنظیمات را به جای ویرایش دستی فایل TOML، مستقیماً درون خود برنامه bgscan تغییر دهید.
>
> برای پیکربندی تعاملی این گزینه‌ها با استفاده از بخش مدیریت تنظیمات (TUI inspector)، از منوی اصلی به مسیر **Settings** ← **ICMP Settings** بروید.

فایل پیکربندی: `settings/icmp_settings.toml`

این فایل پارامترهای مربوط به تنظیم دقیق پروب (بررسی) ICMP را کنترل می‌کند.

## مرجع سریع (Quick Reference)

| تنظیمات | مقدار پیش‌فرض | توضیحات |
|---------|---------|-------------|
| `timeout` | `2000` | حداکثر زمان انتظار (به میلی‌ثانیه) برای دریافت پاسخ ICMP (پینگ). اهدافی که در این بازه پاسخ ندهند، غیرقابل دسترس در نظر گرفته می‌شوند. |
| `tries` | `1` | حداکثر تعداد دفعات تلاش مجدد برای هر هدف جهت کاهش اثر افت پکت (Packet Loss). مقادیر بالاتر دقت را بالا می‌برند اما زمان کل اسکن را افزایش می‌دهند. |
| `workers` | `400` | تعداد ورکرهای هم‌زمان (Concurrent) برای ارسال درخواست‌های پینگ. مقادیر بالاتر سرعت اسکن را افزایش می‌دهند، اما مصرف پردازنده و پهنای باند شبکه را بالا می‌برند. |
| `prefix_output` | `"icmp_"` | پیشوند نام فایل برای تمام خروجی‌های تولید شده توسط این ماژول. این گزینه برای تفکیک نتایج در زمان اجرای هم‌زمان چند اسکن کاربرد دارد. |

## مهلت زمانی (Timeout)

```toml
timeout = 2000

```

حداکثر زمان انتظار (به میلی‌ثانیه) برای دریافت پاسخ ICMP echo reply. اهدافی که در این بازه پاسخ ندهند، غیرقابل دسترس در نظر گرفته می‌شوند.

## دفعات تلاش مجدد (Retries)

```toml
tries = 1

```

حداکثر تعداد دفعات تلاش مجدد برای هر هدف جهت جلوگیری از خطای ناشی از افت پکت (Packet Loss). مقادیر بالاتر قابلیت اطمینان اسکن را بهبود می‌بخشند اما زمان کل اسکن را طولانی‌تر می‌کنند.

## ورکرها (Workers)

```toml
workers = 400

```

تعداد ورکرهای هم‌زمان برای ارسال درخواست‌های ICMP echo request. مقادیر بالاتر سرعت اسکن را افزایش می‌دهند اما منابع بیشتری از CPU و شبکه را درگیر خواهند کرد.

## پیشوند خروجی (Prefix Output)

```toml
prefix_output = "icmp_"

```

پیشوند اعمال‌شده روی نام تمامی فایل‌های خروجی که توسط این ماژول تولید می‌شوند. این ویژگی برای جداسازی و تشخیص نتایج هنگامی که چندین اسکن را به صورت هم‌زمان اجرا می‌کنید، بسیار مفید است.
