# اسکنر

اسکنر bgscan یک موتور اسکن ماژولار و چندمرحله‌ای است که انواع مختلف پروب یا ارزیاب (مانند ICMP، TCP، HTTP، DNS و Xray) را برای بررسی آی‌پی‌ها جهت یافتن سرویس‌ها و آسیب‌پذیری‌های مختلف مدیریت و هماهنگ می‌کند. هر نوع اسکن می‌تواند به صورت کاملاً مستقل اجرا شود یا اینکه در یک خط لوله (Pipeline) به هم متصل شده تا یک فرآیند شناسایی جامع را در شبکه انجام دهد.

برای اطلاعات دقیق‌تر درباره بخش‌های مختلف سیستم اسکن، صفحات زیر را بررسی کنید:

- [انواع اسکن (Scan Types)](./scan-types/)
- [لیست‌های آی‌پی - ورودی (IP Lists)](./ip-files/)
- [فایل‌های نتایج - خروجی (Result Files)](./result-files/)
- [خط لوله اسکن (Scan Pipeline)](./scan-pipeline/)
