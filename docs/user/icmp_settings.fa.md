## تنظیمات ICMP

این تنظیمات رفتار اسکنر ICMP را کنترل می‌کنند از مدت انتظار برای دریافت پاسخ گرفته تا تعداد تلاش‌ها و میزان همزمانی ارسال درخواست‌ها.

---

### `timeout`

```toml
timeout = 2000
```

حداکثر مدت زمان انتظار برای دریافت پاسخ ICMP از هر هدف، بر حسب **میلی‌ثانیه**.

اگر در این بازه زمانی پاسخی دریافت نشود، هدف به‌عنوان **غیرقابل دسترس (Unreachable)** در نظر گرفته می‌شود.

| مقدار  | رفتار                        |
| ------ | ---------------------------- |
| `1000` | انتظار تا ۱ ثانیه            |
| `2000` | انتظار تا ۲ ثانیه (پیش‌فرض)  |
| `5000` | مناسب شبکه‌های کند یا دوردست |

> افزایش این مقدار می‌تواند تعداد پاسخ‌های دریافت‌شده را بیشتر کند، اما زمان تکمیل اسکن را نیز افزایش می‌دهد.

---

### `tries`

```toml
tries = 1
```

تعداد دفعاتی که برای هر هدف درخواست ICMP ارسال می‌شود.

اگر تلاش اول با شکست مواجه شود، اسکنر تا سقف تعیین‌شده دوباره تلاش خواهد کرد.

| مقدار | رفتار                                |
| ----- | ------------------------------------ |
| `1`   | فقط یک تلاش (پیش‌فرض)                |
| `2`   | یک تلاش مجدد در صورت عدم دریافت پاسخ |
| `3+`  | پایداری بیشتر در شبکه‌های ناپایدار   |

> افزایش تعداد تلاش‌ها می‌تواند دقت نتایج را بهبود دهد، اما مدت زمان اسکن را نیز افزایش می‌دهد.

---

### `workers`

```toml
workers = 1000
```

تعداد Workerهای همزمان که وظیفه ارسال درخواست‌های ICMP را بر عهده دارند.

مقادیر بالاتر باعث افزایش سرعت اسکن می‌شوند، اما مصرف CPU، حافظه و پهنای باند شبکه را نیز افزایش می‌دهند.

| مقدار   | رفتار                                 |
| ------- | ------------------------------------- |
| `100`   | مناسب سیستم‌های ضعیف                  |
| `500+`  | تعادل مناسب بین سرعت و مصرف منابع     |
| `1000+` | مناسب سرورهای قدرتمند و اسکن‌های حجیم |

> مقدار بهینه به منابع سیستم و ظرفیت شبکه بستگی دارد.

---

### `prefix_output`

```toml
prefix_output = "icmp_"
```

پیشوندی که به نام فایل‌های خروجی تولیدشده توسط این اسکنر اضافه می‌شود.

این گزینه هنگام اجرای چندین نوع اسکنر به‌صورت همزمان، برای تفکیک فایل‌های خروجی مفید است.

| مقدار       | نمونه خروجی          |
| ----------- | -------------------- |
| `"icmp_"`   | `icmp_results.txt`   |
| `"ping_"`   | `ping_results.txt`   |
| `"custom_"` | `custom_results.txt` |

> تغییر این مقدار روی فرآیند اسکن تأثیری ندارد و فقط نام فایل‌های خروجی را تغییر می‌دهد.
